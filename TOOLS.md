# Tools

Tools and PPE needed for the build. **Parts and consumables are in
`SHOPPING.md`** — that's where steel, hardware, paint, fasteners, and the
cable hardware live.

Checked items are already on hand. Unchecked items still need to be
acquired.

## Welding

- [x] Flux-core / stick welder (entry-level)
- [x] Auto-darkening welding helmet
- [x] Welding gloves (heavy leather)
- [x] Welding jacket or sleeves
- [x] Magnetic welding squares, set of 4 — *critical for end-frame squareness*
- [x] C-clamps and locking pliers
- [x] Chipping hammer
- [x] Wire brush
- [x] Fire extinguisher (ABC) or large spray bottle of water
- [x] Flat work surface — thick scrap plate, flat concrete, or welding table

## Cutting and grinding

- [x] Angle grinder
- [x] Thin cutoff wheels for angle grinder
- [ ] Grinding wheels (1/4" thick) for weld cleanup
- [x] Flap discs for blending welds and finishing
- [x] Wire wheel attachment for de-rusting and prep

## Drilling and tapping

- [x] Drill press
- [ ] 5/16" drill bit for steel — cobalt or HSS, quality bit (used 24+ times)
- [ ] 7/16" drill bit for steel (clearance for 3/8" bolts)
- [ ] 1/2" drill bit for steel (leg back-wall relief holes)
- [ ] 3/8"-16 plug tap (Irwin Hanson or similar — cheap taps break and need to be drilled out)
- [ ] 1/2"-13 plug tap + 27/64" tap drill (Bosch BDT12F13 combo) — for the 4 leveler foot-cap plates
- [ ] Tap handle (T-handle) sized to fit up to a 1/2" tap shank (e.g., 1/4"–3/4" adjustable) — must hold both the 3/8" and 1/2" taps
- [ ] Center punch (or automatic center punch — easier for steel)
- [ ] Cutting fluid (Tap Magic, Rapid Tap, etc.)

## Measuring and marking

- [x] Combination square
- [x] Framing square (12"×24")
- [x] Speed square
- [ ] Soapstone holder + refills (pencils don't show on dark steel)
- [x] Digital calipers (6")
- [x] Bubble level (24" or 4')
- [x] Tape measure

## Hand tools

- [x] 9/16" socket and ratchet (fits 3/8" hex bolts) — part of socket set
- [ ] Torque wrench, 3/8" drive, 5–50 ft-lb range
- [x] Bastard mill file, 8–10" flat (for deburring)

## Woodworking

- [x] Woodworking shop (planer, jointer, table saw, router, drill, etc.)
- [x] Wood stock for the top
- [x] Forstner bits or brad-points sized for threaded inserts in the wood top

## Nice but not required

- [ ] Die grinder + carbide burr (fastest way to make slotted holes for wood-top movement)
- [ ] Cable cutter — Felco or similar (only needed for cable-X version; borrow if possible)

## Steel stock

- [ ] 1.5"×1.5"×0.120" wall A500 square tube — ~50 linear ft (3× 20-ft sticks recommended)
- [ ] 3/4"×3/4"×1/8" mild steel angle — ~14 linear ft (one 20-ft stick)
- [ ] 3/8" mild steel plate stock — ~1 sq ft total (12 plates × 1.5"×5")
- [ ] 3/16" mild steel plate stock — ~1 sq ft total (12 plates × 1.5"×5")

## PPE

- [x] Safety glasses (wraparound, ANSI Z87.1+)
- [x] Cut-resistant gloves (for handling fresh-cut tube ends)
- [x] Hearing protection (earmuffs or plugs)
- [x] Steel-toed boots (optional)
